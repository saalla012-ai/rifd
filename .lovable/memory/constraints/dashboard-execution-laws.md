---
name: Dashboard execution laws
description: قوانين تنفيذ عليا لإعادة هيكلة Dashboard: Campaign Studio، Overview، حدود الاستخدام، وفصل Landing Page.
type: constraint
---
أثناء تنفيذ خطة إعادة هيكلة Dashboard يجب الالتزام بالقوانين التالية دون خرق:

1. Campaign Studio هو مركز قيادة للحملة وليس أداة توليد مباشرة. يستقبل الهدف، يولّد/ينظّم Brief، ثم يوجّه المستخدم إلى أدوات النص والصورة والفيديو كوجهات تنفيذ.

2. Dashboard Overview يجب أن تكون حيوية، موجزة، ومركزة على قرارات سريعة. Hero والبطاقة الذهبية للحملات هما الأهم. ممنوع تحويلها إلى قائمة طويلة من البطاقات؛ استخدم Progressive Disclosure.

3. أرقام حدود النصوص والصور مقاييس تشغيلية داخلية فقط. تظهر في صفحة الاستخدام وشريط الرصيد بصياغة مثل "استخدامك اليومي للنصوص"، ولا تظهر كأرقام بيع أو نقاط تسويقية في التسعير أو الاشتراك أو CTAs الخارجية.

4. Landing Page مستقلة تماماً عن خطة Dashboard. لا تعدل ملفات الصفحة الرئيسية أو صفحات التسويق أثناء تنفيذ خطة Dashboard إلا بطلب صريح.
