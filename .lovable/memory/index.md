# Project Memory

## Core
نطاق البريد المُفعَّل: `send.rifd.site` (الجذر `rifd.site`). المرسل الفعلي `noreply@notify.send.rifd.site`. في الكود اكتب `SENDER_DOMAIN = "send.rifd.site"` فقط — Lovable يضيف `notify` تلقائياً.
خطة Dashboard منفصلة عن Landing Page؛ لا تعدل ملفات الصفحة الرئيسية أثناء تنفيذ الداشبورد إلا بطلب صريح.
Campaign Studio مركز قيادة/Brief وتوجيه، وليس صفحة توليد مباشرة مثقلة.
أرقام حدود النصوص والصور تشغيلية داخلية فقط؛ ممنوع استخدامها كنقاط بيع في التسعير أو CTAs.
