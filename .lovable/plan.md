

## فهم الموقف الاستراتيجي

**عقل (3aqel.com)** = منافس مباشر لرِفد في السوق السعودي، لكن يقدم **ميزة فيديو AI** غير موجودة في رِفد.

بناءً على الفيديوهين المرفوعَين سابقاً + تحليل الموقع، هذه فجوة تنافسية حرجة يجب سدّها.

## التحليل المُنجَز حتى الآن

| ما نعرفه | المصدر |
|---|---|
| عقل يقدم توليد فيديو AI بالعربية | الفيديوهان المرفوعان |
| سعره 27$/شهر تقريباً | تحليل الموقع السابق |
| رِفد حالياً: نص + صور فقط | كود `dashboard.generate-image.tsx` و `dashboard.generate-text.tsx` |
| لا يوجد route لتوليد الفيديو في رِفد | فحص `src/routes/` |

## ما أحتاج تنفيذه (3 مراحل)

### المرحلة 1 — كشف الأدوات التقنية لعقل (Reverse Engineering)
استخدام `ffprobe` على الفيديوهين المرفوعين لاستخراج:
- **Encoder signature** (يكشف المزود: Veo3 / Sora / Kling / Pika / Runway / fal.ai)
- **Codec, resolution, FPS, bitrate, aspect ratio, duration**
- **Color primaries** (بصمة لونية مميزة لكل مزود)
- **Metadata tags** (أحياناً تحتوي اسم المزود صراحةً)

استخراج 8 إطارات مفتاحية لكل فيديو + فحصها بصرياً للبحث عن:
- Watermarks (Sora, Kling, Pika لها علامات مميزة)
- جودة الـ lip-sync (إن وُجد → HeyGen/D-ID)
- جودة النص العربي على الشاشة (مؤشر قوي على المزود)

### المرحلة 2 — التقرير المقارن الاستراتيجي

**جدول استنتاج المزود:**

| الفيديو | المزود الأكثر احتمالاً | درجة الثقة | المؤشرات | التكلفة التقديرية/فيديو |
|---|---|---|---|---|
| فيديو 1 | ... | عالي/متوسط | encoder + style | $X |
| فيديو 2 | ... | ... | ... | $X |

**خريطة الأدوات المُستنتجة لمنظومة عقل:**
- مزود توليد الفيديو الأساسي
- مزود الصوت العربي (ElevenLabs / OpenAI TTS)
- مزود الـ Avatars (إن وُجد)

### المرحلة 3 — توصية استراتيجية لرِفد

تقرير مهني يجيب 3 أسئلة حاسمة:

**1. ما المزود الأنسب لإضافة "توليد فيديو" لرِفد؟**

| الخيار | الإيجابيات | السلبيات | التكلفة |
|---|---|---|---|
| **fal.ai** (aggregator) | عشرات النماذج، API موحّد، أرخص | يحتاج إدارة كرديت | $0.05-0.50/فيديو |
| **Replicate** | مرونة، مكتبة ضخمة | أبطأ قليلاً | مشابه |
| **Veo3 (Google) مباشر** | جودة أعلى | API مُقيّد + غالي | $0.50-2/فيديو |
| **Kling مباشر** | متخصص بالحركة | API صيني، توثيق محدود | $0.30/فيديو |
| **Lovable AI Gateway** | بدون API key، مدفوع تلقائياً | لا يدعم الفيديو حالياً | — |

**2. كيف نتفوق على عقل بدلاً من تقليده؟**
- إضافة **ذاكرة المتجر** للفيديو (نفس ميزة رِفد للنص) — عقل ما عنده
- توليد فيديو **بنبرة سعودية** عبر TTS عربي (ElevenLabs `scribe_v2` متاح)
- تكامل مباشر مع سلة/زد لاستخدام صور المنتج كأساس للفيديو

**3. ما الخطة التنفيذية؟**

ميزة فيديو رِفد V1 (MVP):
- Route جديد: `dashboard.generate-video.tsx`
- يستخدم fal.ai (الأرخص + الأسرع للـ MVP)
- إنتاج فيديو 5 ثوانٍ (مربّع 1:1 أو رأسي 9:16) من نص + صورة منتج
- تكلفة متوقعة: $0.10-0.30/فيديو → سعر بيع داخلي: 3-5 توليدات/شهر للخطة المدفوعة

## نطاق هذا الـ Plan

**ما سأنفّذه بعد الموافقة:**
1. تحليل ffprobe + إطارات + تقرير استنتاج المزود (تحليل بحت، لا تعديلات كود)
2. توصية استراتيجية مكتوبة بالعربية مع 3 خيارات تنفيذية
3. **لا أبني ميزة الفيديو الآن** — الميزة نفسها قرار منفصل يحتاج موافقتك بعد قراءة التقرير

**المخرج:**
تقرير مهني واحد في الشات يجيب: ما الأدوات؟ كم التكلفة؟ ما خطة رِفد للرد التنافسي؟

**لا تعديلات على ملفات المشروع في هذه المرحلة.**
